# Document Archive

This directory holds superseded, historical, or one-off documentation that is
no longer part of Holon's primary documentation surface.

Archive candidates include:

- obsolete roadmap snapshots
- implementation handover notes tied to a past thread
- one-off analysis reports
- fix write-ups that were folded back into runtime docs or code
- older contract drafts that have been replaced by canonical RFCs in
  `docs/rfcs/`

Rules:

- archived documents may still be useful for historical context
- archived documents should not be treated as the current product contract
- new architecture and runtime contract work should land in `docs/rfcs/`
  instead of adding new top-level historical notes
- archived documents may preserve historical cross-references and are not
  maintained as the primary navigation surface
