# Archived Documentation

This directory contains historical documentation that has been superseded by
canonical RFCs or implementation decisions.

These documents are preserved for historical reference but should not be
treated as current Holon contracts. For canonical architecture and design
decisions, see:

- `../rfcs/` - Canonical architecture contracts
- `../implementation-decisions/` - Historical ADR-style records

## Archive Categories

### Superseded by RFCs
Documents that were replaced when RFCs became the canonical source of truth.

### Historical Planning
Old roadmaps, planning notes, and direction documents that captured
decision-making context but are no longer active planning documents.

### Reference Material
Comparative analysis and research notes that may be useful for context
but are not normative Holon documentation.
